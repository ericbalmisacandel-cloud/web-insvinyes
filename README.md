## Projectes de programació

### Chatbot FIFA (Cris)

Projecte desenvolupat en Python.

És un chatbot interactiu que:
- Dona consells sobre jugadors i equips
- Fa prediccions
- Explica acudits
- Manté una conversa amb l’usuari

Objectiu del projecte:  
Aprendre estructures de control, funcions i ús de la llibreria `random` en Python.

### Pàgina Web FIFA

Projecte desenvolupat amb:
- HTML
- CSS
- JavaScript

Inclou:
- Disseny estil videojoc
